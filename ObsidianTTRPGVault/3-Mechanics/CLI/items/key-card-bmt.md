# Key Card
*Wondrous item, legendary*  
![](file://decks/BMT/Deck%20of%20Many%20Things/02-key.webp#right)


As an action, you can brandish this card and use it to cast the [Knock](3-Mechanics/CLI/spells/knock-xphb.md) spell, with no sound made by the spell. This property can be used five times, and it regains all expended uses at dawn.

*Source: The Book of Many Things p. 14*
