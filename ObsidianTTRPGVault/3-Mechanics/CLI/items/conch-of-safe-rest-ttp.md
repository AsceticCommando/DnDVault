# Conch of Safe Rest


- **Weight**: 2.0 lbs.

You can use an action to blow this pearlescent conch like a [horn](3-Mechanics/CLI/items/horn-xphb.md), creating an effect identical with that of a [Leomund's tiny hut](3-Mechanics/CLI/spells/leomunds-tiny-hut-xphb.md) spell. The conch disappears once the spell takes effect. The effect ends if you leave the spell's area.

*Source: The Tortle Package p. 17*
