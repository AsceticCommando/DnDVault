- **bpdc.** back post double crochet  
- **bphdc.** back post half double crochet  
- **blo.** back loop only  
- **ch.** chain  
- **ch sp.** chain space  
- **dblpuff.** double puff stitch  
- **dc.** double crochet  
- **dcdec.** double crochet decrease (also called dc2tog)  
- **dcinc.** double crochet increase (2dc in same stitch)  
- **dcXtog in the same stitch.** double crochet (#) stitches in the same stitch  
- **dc3tog.** double crochet 3 stitches together  
- **dc5tog.** double crochet 5 stitches together  
- **dec.** single crochet decrease (also called sc2tog)  
- **fdc.** foundation double crochet  
- **flo.** front loop only  
- **fpdc.** front post double crochet  
- **fptr.** front post treble crochet  
- **fsc.** foundation single crochet  
- **hdc.** half double crochet  
- **hdc dec.** half double crochet decrease (also called hdc2tog)  
- **hdc inc.** half double crochet increase (2hdc in same stitch)  
- **htr.** half treble crochet (U.S. version)  
- **inc.** single crochet increase (2sc in same stitch)  
- **inv dec.** invisible decrease  
- **picot.** picot stitch  
- **popcorn.** popcorn stitch  
- **puff st.** puff stitch  
- **sc.** single crochet  
- **sk st.** skip a stitch  
- **slst.** slip stitch  
- **tr.** treble crochet  
- **Vstitch.** 2dc in same stitch  

## Special Stitches

- **2dec in 3sc.** Make a standard decrease, then start in the same stitch you just decreased into and make another decrease. This results in 2 stitches (2dec) where there were 3sc.  
- **Back Loop Only (blo).** Work through back loop of a stitch only.  
- **Back Post Half Double Crochet (bphdc).** Yarn over and insert the hook from the back to front to back around the post of the stitch of the previous row, yarn over and pull up a loop, yarn over and pull through all 3 loops on your hook.  
- **Cable.** sk 2 sts, fptr in each of next 2 sts, fptr in each of the 2 previously skipped stitches. (1 cable is 4 stitches)  
- **Crab Stitch / Reverse Single Crochet (crab st).** Ch1. Without turning your work, insert the hook from front to back in the first single crochet stitch. Yarn over your hook and draw up a loop. Yarn over and draw it through both loops.  
- **DcXtog in the same stitch / Bobble Stitch.** This is often called a "bobble stitch," but this book uses "dcXtog in the same stitch" because it allows you to vary the number of dc5tog to make smaller or larger bobbles, as needed. For example, to dc3tog in the same stitch, (1) yarn over, insert hook into stitch, yarn over and pull up a loop, yarn over and pull through two loops (leaving 2 on hook); (2) yarn over, insert hook into same stitch, yarn over and pull up a loop, yarn over and pull through 2 loops (leaving 3 on hook); (3) yarn over, insert hook into the same stitch, yarn over and pull up a loop, yarn over and pull through 2 loops, yarn over and pull through all 4 loops on hook to finish.  
- **Double Puff Stitch (dblpuff).** Yarn over, insert hook into first space and pull up a loop 2 times, yarn over and insert hook into next space 3 times, yarn over and insert hook into next space 2 times, yarn over and pull through all loops on the hook, and ch1. To work the next row above the dblpuff, work in the ch1 space. You are decreasing when you make this stitch.  
- **Front Loop Only (flo).** Work through front loop of a stitch only.  
- **Half-Triple or Half-Treble Crochet (htr).** Yarn over twice, insert hook into stitch, pull up a loop, pull through 2 loops on the hook, yarn over, pull through remaining 3 loops.  
- **Invisible Decrease (inv dec).** Insert hook into front loop of indicated stitch, insert hook into front loop of next stitch (3 loops on hook). Yarn over, pull through 2 loops. Yarn over, pull through remaining 2 loops on hook.  
- **Picot Stitch (picot).** Ch2 or 3, depending on the pattern, slst into the front loop of the last st made.  
- **Popcorn Stitch (popcorn).** 5dc in stitch indicated, remove hook from loop, insert hook into first dc made, grab loop and pull through, ch1 to finish. When working in the top of the popcorn, work in the ch1.  
- **Puff Stitch (puff st).** Yarn over, insert hook and pull up a loop 5 times, yarn over and pull through all loops on hook and ch1. To work the next row above the puff, work in the ch1 space.  
- **Scale / Crocodile Stitch (scale).** Working around the designated dc stitches, work 5dc around the post of the first dc, ch1, work 5dc around the post of the next dc (1 scale made).  
- **Spike Stitch (spike st).** This stitch is worked using the same steps as a single crochet stitch, but instead of working into the current row where you are crocheting, you drop down and work it into a row below that current row.  
- **Vstitch.** 2dc in the same stitch.
